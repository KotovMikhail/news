Технология Power
Mapping

  • Первый день.
    Форсайт-сессия.     Мы определим видение будущего компании и четко зафиксируем задачи этого года - чтобы вы понимали, куда идти на постоянно меняющемся рынке.

    • Определяем куда движемся

      Силы и достижения / Проблемы и боли / Бизнес-цели на год / Задач на год с приоретизацией (must have, should have, nice to have)

    • Характеризуем что внутри

      Продукты (звезды, собаки, коровы) / Каналы / Партнеры / Клиенты / Команда

    • Анализируем что вокруг

      Рынок: ситуация, тренды / Конкуренты: лидеры и новые звезды / Люди: консьюмер тренды

  • Второй день. Сессия «Сила Бренда». Мы определим и структурируем план работы - чтобы бренд 365 дней в году помогал достигать поставленных целей.

    • Проводим аудит бренда

      Позиционирование / Архитектура / Вербальная платформа / Визуальная платформа / Текущие коммуникации / Состояние процессов внедрения

    • Определяем территории усиления

      Для разработки / Для коррекции / Для операционных доказательств / Для коммуникации

    • Разрабатываем план действий и составляем чеклист

      Действия / KPI’s / Ресурсы / Бизнес-эффект / Риски / Скорость внедрения
